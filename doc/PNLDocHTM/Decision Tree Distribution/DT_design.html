Class CTreeCPD Class CTreeCPD containes implementations of Class CCPD virtual functions.

Methods that provides inference:
UpdateTreeUpdates decision Tree for current node distribution function.
ConvertWithEvidenceToPotentialCreates new Tabular or Gaussian potential using current evidence on decision Tree parents.

Class CTreeDistribFun Class CTreeDistribFun implements basic operations with distribution on Desidion Tree node. In this case distribution can de defined by user.

Methods that provides inference:
UpdateTreeUpdates Deisgion Tree.
ConvertGraphToTreeCreates a tree structure by graph whithout filing nodes properties
FillTreeFills tree nodes properties.
SetDomainAllows to define parents for current node.
GenerateSampleGenerates sample using current evidence.
GetAdjectivesIn continuous case returns correspound expectation and covariance using current evidence.
GetProbabilityIn discrete case returns correspound probability vector using current evidence.
