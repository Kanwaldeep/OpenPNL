Class CTtreeCPD

Class CTreeCPD containes implementations of Class CCPD virtual functions.

UpdateTree

Updates decision Tree for current node distribution function.

void UpdateTree(const CGraph *pGraphTree, TreeNodeFields *fields);
Arguments
pGraphTree Pointer to the class CGraph that will be used for creation a new decision Tree.
fields Pointer to the array of the new decision Tree nodes properties.

ConvertWithEvidenceToPotential

Creates new Tabular or Gaussian potential using current evidence on decision Tree parents.

CPotential* ConvertWithEvidenceToPotential(const CEvidence* pEvidence, int flagSumOnMixtureNode = 1) const;

Arguments
pEvidence Pointer to the current decision Tree parents evidence.
flagSumOnMixtureNode Flag of mixture node summation.
