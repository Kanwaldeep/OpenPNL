Class CTreeDistribFun

Class CTreeDistribFun implements basic operations with distribution on Desidion Tree node. In this case distribution can de defined by user.

UpdateTree

Updates decision Tree structure for node distribution function.

void UpdateTree(const CGraph* pGraphTree, TreeNodeFields *fields);
Arguments
pGraphTree Pointer to the class CGraph that will be used for creation a new decision tree.
fields Pointer to the array of the new decision Tree nodes properties.

ConvertGraphToTree

Creates new decision Tree structure without filling tree nodes properties.

void ConvertGraphToTree(CxCART *pCart,CxCARTNode *node, const CGraph *pGraph, int nodeNum) const;
Arguments
pCart Pointer to the structure that allows storing decision Tree.
node Pointer to the stucture that allows storing tree node properties.
pGraph Pointer to the class CGraph that will be used for creation a new decision tree.
nodeNum Current added node number.

FillTree

Fills tree nodes properties.

void FillTree(CxCART *pCart,CxCARTNode *node, const CGraph *pGraph, TreeNodeFields *fields, int nodeNum) const;
Arguments
pCart Pointer to the structure that allows storing decision Tree.
node Pointer to the stucture that allows storing tree node properties.
pGraph Pointer to the class CGraph that will be used for creation a new decision tree.
fields Pointer to the array of the new decision Tree nodes properties.
nodeNum Current added node number.

SetDomain

Allows to define parents for current node.

void SetDomain(intVector domain );
Arguments
domain Node domain.

GenerateSample

Generates sample using current evidence.

void GenerateSample(CEvidence* evidence, int maximize) const
Arguments
evidence Pointer to the current decision Tree parents evidence.
maximize Flag of the marginalization type:
1)for discrete variables:
-0 stands for simple summation.
-1 stands for finding maximum value.
2)for continuous variables:
both are integration operations..

GetAdjectives

In continuous case returns correspound expectation and covariance using current evidence.

int GetAdjectives(const CEvidence* pEv, float &expect, float &varian) const;
Arguments
pEv Pointer to the current decision Tree parents evidence.
expect decision Tree node expectation for current parents configuraion.
varian decision Tree node variance for current parents configuraion..

GetProbability

In discrete case returns correspound probability vector using current evidence.

floatVector GetProbability(const CEvidence* pEv) const;
Arguments
pEv Pointer to probability vector for current evidence.
