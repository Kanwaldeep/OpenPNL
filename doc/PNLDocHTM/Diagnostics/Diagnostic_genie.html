Installation instructions

To use PNL with GeNIe interface install GeNIe on your computer (only for Windows).

  1. Run the installer genie2_0_1864_0.exe from PNL\high\PNLGeNIe\GeNIe\ and follow the instructions.
  2. Copy to installation directory all dll from PNL\high\PNLGeNIe\GeNIe\.

Note

There are several limitations on networks and Diagnostics algorithm in GeNIe using PNL.

  • Limitations on network - see PNL/high/doc/WrappersUserGuide.htm#Limitations
  • At present only two algorithms are realized in PNL (Marginal/Strength 1, Marginal/Strength 2).
