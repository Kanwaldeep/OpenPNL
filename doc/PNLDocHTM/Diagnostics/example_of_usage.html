Example of usage diagnostics with Bayesian networks

Document contains necessary information to start a simple test in diagnostic algorithm for Bayesian networks that are realized in PNL. At first simple network is presented. Further an example for this algorithm is described.

Initial model

Consider the following network also known as Asia network.


This network consist of 8 discrete nodes. Node sizes are 2. 0,1,6,7 is test nodes. 2,3,4 is target nodes.

Implementation
To start diagnostics with this network you may use the following source code. 
int main(int argc, char* argv[])
{
/*---- Network creation -----------------------------------------------------*/
    const CBNet* pBNet;
    pBNet = pnlExCreateAsiaBNet();   // load standart model example 

/*---- Diagnostics creation -------------------------------------------------*/
    CDiagnostics *pDiag = CDiagnostics::Create(pBNet);
    pDiag->SetCostRatio(1);          // set parameter for VOI function

/*---- Setting of target nodes ----------------------------------------------*/
    intVector tarN;
    tarN.push_back(2);
    tarN.push_back(3);
    tarN.push_back(4);
    pDiag->SetTargetNodes(tarN);

/*---- Setting of test nodes ------------------------------------------------*/
    intVector testN;
    testN.push_back(0);
    testN.push_back(1);
    testN.push_back(6);
    testN.push_back(7);
    pDiag->SetObservationNodes(testN);
    pDiag->SetCost(0,1);             // set cost of observation for 0 node
    pDiag->SetCost(1,1);             // set cost of observation for 1 node

/*---- Setting of target states ---------------------------------------------*/
    pDiag->SetTargetState(2,1);
    pDiag->SetTargetState(3,1);
    pDiag->SetTargetState(4,1);

/*---- Choosing of selecting function ---------------------------------------*/
    pDiag->SetAlgorithm(0); 

/*---- Vector of pursued node creation --------------------------------------*/
    intVector purN;
    purN.push_back(2);
    purN.push_back(3);
    purN.push_back(4);
    intVector purS;
    purS.push_back(1);
    purS.push_back(1);
    purS.push_back(1);
/*---- Diagnostics execution ------------------------------------------------*/
    intVector res = pDiag->GetTestsList(purN,purS);
/*---- Result output --------------------------------------------------------*/
    int i;
    for( i = 0; i < res.size(); ++i )
    {
        printf("%d\n", res[i]);
    }
    return 0;
}
