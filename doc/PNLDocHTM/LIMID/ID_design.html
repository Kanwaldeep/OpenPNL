Design


Class CIDPotential implements basic operations with factors. This class is an abstract one, so it can’t be used itself.

Operations that can be performed with a CIDPotential object are as follows:

  • combination and division of CIDPotential objects
  • contraction of CIDPotential objects
  • marginalization

Methods of this class are:
CombineCombines two factors and returns product
ContractionMultiplies utility and probability parts of factor and returns the product
DivideInSelfProvides division and puts the result to potential
InsertPoliticsInSelfInserts new politics to the current CIDPotential object
SetProbDistribFunSets the probability part of potential
SetUtilityDistribFunSets the utility part of potential

Class CIDTabularPotential contains implementations of class CIDPotential virtual functions. This class is used in inference algorithm for the nets, which are based on the Limited Memory Influence Diagram.

Methods of this class are:
CreateReturns CIDTabularPotential class object
CloneCreates a new CFactor class object as copy of the input object

Class CIDNet represents Influence Diagrams. This class is a descendant of CBNet class. We can consider Influence Diagrams as a specific case of Bayesian networks with three different types of nodes.

Methods of this class are:
CreateReturns CIDNet class object
GetUnValueNodesGets chance and decision nodes of the input object
GetDecisionNodesGets decision nodes of the input object.
GetValueNodesGets utility nodes of the input object
IsLIMIDChecks whether the net is Limited Memory Influence Diagram or not

Class CLIMIDInfEngine is a main class for carrying out an inference in Limited Memory influence Diagrams. This class is a descendant of CPNLBase class because inference process in LIMIDs is not the same with such process in Bayesian networks.

Methods of this class are:
CreateReturns CLIMIDInfEngine class object
ReleaseDestroys the inference engine
SetIterMaxSets maximum quantity of iterations
DoInferenceMain method, which does inference in Limited Memory Influence Diagrams
GetPoliticsGets optimal politics after the inference
GetExpectationGets expectation after the inference
