Class CSoftMaxCPD

Class CSoftMaxCPD containes implementations of Class CCPD virtual functions.

Create

Returns class object

static CSoftMaxCPD* Create(const int *domain, int nNodes, CModelDomain* pMD);
Arguments
domain Pointer to the array of numbers of domain nodes.
nNodes Number of nodes in domain.
pMD pointer to model domain, which includes this factor

Copy

Creates new SoftMax CPD as copy of the input object

static CSoftMaxCPD* Copy(const CSoftMaxCPD* pSMCPD);
Arguments
pSMCPD Pointer to CSoftMaxCPD object.

AllocDistribution

Allocates SoftMax distribution on SoftMax child node.

void AllocDistribution(const float* pWeights, const float* pOffsets, const int* parentCombination = NULL);
Arguments
pWeights Pointer to SoftMax weight matrix.
pOffsets Pointer to SoftMax offset vector.
parentCombination Pointer to discontinous parents state combination.
Discussion
This function allocates a SoftMax distribution on SoftMax child node with Gaussian parents and the given discrete parent combination.

GetMaximizingMethod

Gets method, which is used for likelihood maximizing.

EMaximizeMethod GetMaximizingMethod();
Discussion
There are three possible methods for likelihood maximising. They are Gradient, ConjugateGradient and Hessian

SetMaximizingMethod

Sets method, which is used for likelihood maximizing.

void SetMaximizingMethod(EMaximizeMethod met);
Arguments
met Defines Likelihood maximizing method, which you want to use.

MaximumLikelihood

Maximizes likelihood using defined method.

void MaximumLikelihood(float ** Observation, int NumberOfObservations, float Accuracy, float step = 0.1);
Arguments
Observation pointer to float matrix of observations on the SoftMax node family.
NumberOfObservations Defines quantity of observations
Accuracy Accuracy - parameter of maximizing method
Step Step - parameter of maximizing method
