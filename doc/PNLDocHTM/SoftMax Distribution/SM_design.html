Design


Class CSoftMaxCPD Class CSoftMaxCPD containes implementations of Class CCPD virtual functions.

Methods of this class are:
CreateReturns class object
CopyCreates new SoftMax CPD as copy of the input object
AllocDistributionAllocates SoftMax distribution on SoftMax child node
SetMaximizingMethodSets method, which would be used for likelihood maximizing
GetMaximizingMethodGets method, which is used for likelihood maximizing
MaximumLikelihoodMaximizes likelihood using defined method

Class CSoftMaxDistribFun Class CSoftMaxDistribFun implements basic operations wiht distribution on SoftMax node when it has only continuous parents. In this case distribution can de defined by one matrix of weights and one offset vector.

Methods of this class are:
CreateReturns class object
CopyCreates a copy of an input object and returns a pointer to it
CalculateHessianCalculates Hessian matrix for Likelihood function using Evidences
CalculateLikelihood Calculates Likelihood using Evidences
MaximumLikelihoodGradient
MaximumLikelihoodConjugateGradient
MaximumLikelihoodHessian		Functions for likelihood maximizing

Class CCondSoftMaxDistribFun Class CCondSoftMax implements basic operations wiht distribution on SoftMax node when it has both continuous and descrete parents. In this case distribution can be defined by number of SoftMaxDistribFun objects. Each of these objects corresponds to definite desrete parents state configuration.

Methods of this class are:
CreateCreates a CCondSoftMaxDistribFun class object and returns a pointer to it.
CopyCreates a copy of input object and returns a pointer to it
SetDistribFunSets definite Distribution function as distribution on SoftMax node with definite discrete parents state combination
MaximumLikelihood Calls MaximumLikelihood method for every CSoftMaxDistribFun object
